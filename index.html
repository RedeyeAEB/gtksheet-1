General Information

The gtksheet library is a fork of the GtkSheet widget included in the GTK+extra library. It includes the GtkSheet widget, a widget displaying a spreadsheet-like matrix.

The library is written in C and requires GTK+ version 2.x or 3.x. It has support for Glade and GObjectIntrospection. There is also a Lua LGOB binding available.

NOTE: This fork was reset and updated  on 05.04.2018. The project homepage and code cleanup is still under work. Currently, you will get a GtkExtra package when building, this will be changed to GtkSheet as soon as possible.

Getting gtksheet

You can download the latest gtksheet releases here.

The latest gtksheet release is gtksheet version 0.2.0: gtksheet-0.2.0.tar.gz

Finally, you can download the latest development code from the github.com repository using git:

$ git clone https://github.com/fpaquet/gtksheet.git
